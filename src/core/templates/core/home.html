{% trans 'What is the Web 4D-QSAR?' %}

The 4D-QSAR Web is a system that involves performing molecular dynamics using GRAMACS, and generation of arrays of molecular descriptors 4D, with a new version of the methodology LQTA-QSAR, the LQTAGridPy. Thus, the result is a well-integrated and functional computational package, abstracting the technical steps of the execution of separate applications such as GROMACS and LQTAGridPy, moreover, the fact of the existence of an environment already configured and available on the Internet.
